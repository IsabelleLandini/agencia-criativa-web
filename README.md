# 🚀 Agência Criativa Web

Projeto de landing page desenvolvido com **HTML5 e CSS3**, com foco em prática de layout moderno, responsividade e organização de código.

---

## 📌 Sobre o projeto

A **Agência Criativa Web** é uma landing page fictícia criada para simular o site de uma agência digital.

Nesta versão do projeto foi realizada uma refatoração do código CSS, com foco em organização, reutilização e manutenção, aplicando os conceitos de arquitetura CSS estudados no módulo.

Principais melhorias implementadas:

* Aplicação da metodologia BEM (Block, Element, Modifier)
* Organização do CSS em seções lógicas
* Criação de componentes reutilizáveis
* Redução da especificidade dos seletores
* Eliminação do uso de IDs para estilização
* Melhor manutenção e escalabilidade do código
* Preservação da responsividade em diferentes dispositivos

---

## 🎨 Layout do projeto

O site é composto por:

* 🏠 **Home** – apresentação principal com banner e chamada para ação
* 👩‍💼 **Sobre Nós** – descrição da agência e valores
* 💼 **Serviços** – cards com os principais serviços oferecidos
* 💬 **Depoimentos** – feedback de clientes fictícios
* 📞 **Contato** – formulário e informações de contato

## 🔧 Refatoração CSS

Como parte da atividade de refatoração, o projeto passou por uma reorganização estrutural do CSS, incluindo:

* Padronização da nomenclatura utilizando BEM
* Separação entre layout, componentes e seções específicas
* Criação de classes reutilizáveis como `.button`, `.card` e `.section`
* Redução da duplicação de código
* Melhoria da legibilidade e manutenção dos estilos
* Preservação do comportamento responsivo da aplicação
